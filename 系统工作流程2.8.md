## 二、v2.8 核心优化

### 启动流程 (v2.8)

```
双击 EXE
  │
  ├─ launcher.py 启动 → 注册 console handler → uvicorn 启动
  │
  ├─ FastAPI lifespan:
  │   ├─ 数据库 + 下载管理器 (同步, <0.1s)
  │   ├─ asyncio.create_task(浏览器后台初始化)  ← 不阻塞!
  │   └─ 服务就绪 ← 前端立即可达
  │
  ├─ 前端加载屏 (立即显示):
  │   ├─ 每 500ms 轮询 GET /api/ready
  │   ├─ 首次加载显示 "正在唤醒后端服务"
  │   └─ 浏览器就绪 → 自动进入主界面
  │
  └─ /api/ready 端点:
      {"status": "loading"} → 浏览器初始化中
      {"status": "ready"}   → 全部就绪
```

### 关闭流程 (v2.8)

```
点击控制台 × 或 Ctrl+C
  │
  ├─ SetConsoleCtrlHandler 捕获 CTRL_CLOSE_EVENT
  ├─ os._exit(0) 直接终止 ← 不等待任何清理
  ├─ uvicorn timeout_graceful_shutdown=2
  └─ < 2 秒退出
```

## 三、排查模式

| 模式 | 路由 | 站点 | 后端 | 耗时 |
|------|------|------|------|------|
| **快速排查** (默认) | `/api/fetch` | Google Play + APKPure + APKCombo | Fetcher | ~12s |
| **快速排查** | `/api/fetch/fast` | 同上 | Fetcher | ~12s |
| **慢速排查** | `/api/fetch/slow` | APKMirror + APKVision | StealthySession | 30-90s |
| **慢速异步** | `/api/fetch/slow/async` | APKMirror + APKVision | StealthySession | 后台 |
| **全量排查** | `/api/fetch/all` | 全部 5 个 | Fetcher + Stealthy | 30-100s |
| **多包名排查** | `/api/fetch/batch` | 同上 | Semaphore 限流 | ~12s/3包名 |

## 四、数据提取

### Google Play 双语言 (v2.7)

| 语言 | 用途 | 字段 |
|------|------|------|
| `lang=zh, country=cn` | 标题、版本 | `title`, `version` |
| `lang=en, country=us` | 更新内容、版本补充 | `recentChanges`, `version` |

并行请求 (`asyncio.gather`)，Varies with device 时仍提取 app_name + whats_new。

### APKPure 提取

| 步骤 | 方法 |
|------|------|
| 搜索页 | Fetcher → 正则提取详情 URL (包名路径验证 + 多候选取第一) |
| 详情页 | extract_both(version+vc) + .whats-new-container p.text (深度提取) |
| 中文站 | `_to_cn_url()` 转换 → `apkpure.net` (不重定向) |
| 清洗 | `html.unescape` 解码实体 |

### APKCombo 提取

| 步骤 | 方法 |
|------|------|
| 版本 | `/api/app/{pkg}` → 302 详情页 → extract_both |
| 名称 | `itemprop="name"` / `title` 提取 |
| 更新 | `data-role="whatsnew"` / `whats-new` 提取 |

### 版本合理性校验 (v2.7)

```
is_plausible_version("1.2.183")   → True
is_plausible_version("2.723.787") → True  (格式合法，共识机制会覆盖)
is_plausible_version("2723787")   → False
is_plausible_version("1.2.3.4.5") → False
```

## 五、版本比对

```
版本号对比 (优先):
  compare_version_codes → -1=newer, 0=matched, 1=older

版本名对比 (fallback):
  compare_versions → 含 normalize + semver 支持

app_name 优先级:  Google Play > APKPure > APKCombo
whats_new 优先级:  APKPure > APKCombo > Google Play (全部占位符过滤)

综合状态: 优先 version_code, 其次 version_name
```

### 下载链接

| 源 | 方法 | URL 格式 |
|----|------|---------|
| APKPure | URL 构造 | `apkpure.net/{slug}/{pkg}/download` |
| APKCombo | URL 构造 / Fallback | `apkcombo.com/{slug}/{pkg}/download/phone-{ver}-apk` / `apkcombo.com/{pkg}/download` |
| APKMirror | StealthySession | 动态 |
| APKVision | StealthySession | 动态 |

| 源 | aiohttp | Playwright | 原因 |
|----|---------|-----------|------|
| APKCombo | ❌ | ✅ | 防盗链 |
| APKPure | ❌ | ✅ | d.apkpure.net 强防盗链 |
| APKMirror | ⚠️ | ✅ | CF 保护 |
| APKVision | ⚠️ | ✅ | 同上 |

## 八、批量 Excel

```
上传 → 解析列 → BatchTask → asyncio.create_task
  → Semaphore(5) + as_completed 并发
  → JSONL 流式写盘
  → 写回 Excel: 排查时间 | 版本号(vc) | 对比状态
  → 完成弹窗汇总
  → BatchManager._cleanup_temp()
```
