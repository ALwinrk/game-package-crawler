## v2.6 (2026-06-04)

### 🎨 前端全面美化 (leleo.top 风格)
- **主题系统**: CSS 变量驱动的深色/浅色双主题，支持系统偏好检测 + 手动切换 (localStorage 持久化)
- **毛玻璃效果**: 顶部导航栏 `backdrop-blur` 半透明，动态渐变背景
- **组件重构**:
  - `PackageInput.vue` — 发光聚焦输入框、emjoi 按钮、记忆占位符提示、幽默 Toast
  - `ResultTable.vue` — emoji 状态标签 (✅🆕📦❌)、等宽字体版本号、卡片悬停动画
  - `BatchPanel.vue` — 上传区虚线发光边框、进度条渐变填色、完成弹窗彩色汇总
  - `DownloadQueue.vue` — 卡片化布局替代原始表格、状态左边框色条、交错入场动画
  - `SettingsPanel.vue` — 设置项卡片分组 (网络/并发/重试/站点/GP)、主题开关、代理测试、清除缓存
- **动画系统**: `fadeInUp` / `glowPulse` / `bgShift` / `shimmer` 等 6 个关键帧动画
- **Element Plus 全局主题覆盖**: 按钮渐变、输入框发光、表格行悬停、进度条渐变
- **全局样式**: 新增 `frontend/src/styles/global.css` (350 行 CSS 变量 + 工具类)

### 🔧 APKCombo 中文站 URL 修复
- **根因**: `_extract_slug_from_url()` 对 API 格式 URL (`/api/app/{pkg}`) 错误提取 `slug="api"`
- **修复**: 新增路径黑名单 `{"api","search","download"}`，API URL 返回空字符串
- **Fallback**: `extract_apkcombo_links` 新增兜底 `https://apkcombo.com/zh/{package}/download`
- **URL 格式**: `apkcombo.com/zh/{slug}/{pkg}/download/phone-{ver}-apk`（有 slug）/ `apkcombo.com/zh/{pkg}/download`（无 slug 兜底）

### 📝 游戏中文名 & 更新内容提取
- **数据模型**: `ApkInfo` + `FetchResult` 新增 `app_name`、`whats_new` 字段
- **Google Play**: 切换 `lang=zh, country=cn`，提取 `title` → app_name，`recentChanges` → whats_new（带占位符过滤）
- **APKPure**: 新增 HTML 解析 — `data-dt-title` / `data-dt-whatsnew` 属性提取 + `_clean_html()` 标签清洗
- **APKCombo**: 新增 HTML 解析 — `itemprop="name"` / `data-role="whatsnew"` 属性提取
- **聚合优化**: `app_name` 优先级 GP > APKPure > APKCombo；`whats_new` 优先级 APKPure > APKCombo > GP（全部源过滤占位符 "What's New in the Latest Version" 等无效文本）
- **前端展示**: 结果表格上方新增应用信息卡片（中文名 + 包名 + 更新内容，支持多行纯文本复制）

### 🐛 Bug 修复
- 修复 `BatchTask._cleanup_temp` → `BatchManager._cleanup_temp` (3 处错误引用)
- 修复 `BatchTask._load_results_from_temp` → `BatchManager._load_results_from_temp` (1 处)

## v2.4 (2026-06-03)

- **TTL 爬虫缓存**: `ScraperCache` 内存缓存 (300s TTL) + `POST /api/cache/clear`
- **慢速异步任务**: `POST /api/fetch/slow/async` 非阻塞后台查询 + WebSocket 完成通知
- **批量内部并发**: `batch_concurrency=5`, `asyncio.Semaphore` + `as_completed` 流式处理
- **批量结果流式持久化**: JSONL 临时文件, 降低内存占用
- **批量 Excel 导出增强**: 蓝色表头样式恢复 + 列自动检测复用 + dict 格式兼容
- **下载缓冲区可配**: `download_chunk_size` 默认 1MB
- **SQLite 性能优化**: `synchronous=NORMAL`, `foreign_keys=ON`
- **HTTP 客户端重构**: 三级后端明确分层 (Fetcher / JS Render / StealthySession)
- **APKPure/APKCombo 浏览器下载页**: URL 构造替代 Playwright 捕获
- **前端慢速模式启用**: slow/async 模式可切换
- **ResultTable 修复**: 移除无效 `virtual-scroll`, 固定 `max-height=400`

### 新增配置项
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `batch_concurrency` | 5 | 批量任务内部并发包名数 |
| `cache_ttl_seconds` | 300 | 爬虫结果缓存有效期(秒) |
| `download_chunk_size` | 1048576 | 下载缓冲区字节数 (1MB) |
