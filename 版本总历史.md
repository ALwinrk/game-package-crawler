## v3.4 (2026-06-10) — 增量更新 + 稳定性

### 📊 实时面板增量更新
- **Top-N 增量算法**: 只抓首页新游戏，遇到已知包名提前终止
- **请求量对比**: 全量 ~300 → 增量 ~50（-83%），大幅降低 Cloudflare 风控
- **入库去重**: `(source, package_name)` UNIQUE 索引，INSERT OR REPLACE 自动合并
- **容量控制**: `panel_max_items=150` 数据库上限，`apkpure_display=90` / `apkcombo=90` / `apkvision=60` 分源可配
- **双刷新模式**: 全量刷新（完全覆盖）+ 增量刷新（只追加）+ 刷新面板（仅拉取显示）
- **首次全量**: `full_refresh` 标志跳过提前终止，首次启动强制全量爬取

### 🔧 EXE 稳定性
- **全局异常捕获**: `main()` 包裹 try/except，崩溃时暂停显示错误
- **端口占用检查**: 启动前检测 8000 端口并提示
- **Chromium 三路查找**: 持久目录 → Temp → ms-playwright（服务器 Playwright 兜底）
- **版本号统一**: index.html + App.vue + main.py 全部同步 v3.4

### 📥 下载修复迭代
- **图标过滤**: 严格匹配 `/.apk` 排除 PNG/rs:fit/imgs/no_login
- **JS 触发下载**: Playwright 页面内创建 `<a>` 元素点击，保留 Referer 解决 403
- **浏览器页提前路由**: APKCombo/APKPure 下载页跳过 HEAD+aiohttp 直接 Playwright
- **networkidle→domcontentloaded**: 解决 APKPure 页面加载超时 20s

### 🛡️ 其他修复
- **配置新增**: `panel_max_items`, `apkpure_display_limit`, `apkcombo_display_limit`, `apkvision_display_limit` 热更新支持
- **API 新增**: `POST /api/daily-updates/refresh-incremental` 增量刷新端点
- **熔断修复**: `record_success` 自动恢复默认间隔 1800s
- **前端空 catch**: 全部补充 `console.error` 日志

## v3.3 (2026-06-09) — 反封禁 + 下载修复

### 🛡️ Cloudflare 反封禁四层防护
- **TLS 指纹轮换**: `http_client.py` Chrome 系指纹池 (`chrome110/116/120/124/edge101`)，每次请求随机选取
- **请求随机化**: APKPure 分类顺序随机打乱 + 分类间隔 3-7s 随机暂停 + 单包名延迟 0.5-2s
- **浏览器反检测**: `--disable-blink-features=AutomationControlled` + 随机 viewport (1280-1920×768-1080) + UA 池 (3 个) + `navigator.webdriver` stealth 脚本
- **熔断器增强**: 连续失败 2 次自动降频至 7200s / 成功恢复 1800s / `POST /api/apkpure/unblock` 手工重置接口 (仅本地)

### 📥 下载系统修复
- **APKCombo URL 双语言码修复**: `_extract_slug_from_url` 跳过 20 种语言代码，消除 `/zh/zh/` 双写
- **APKPure URL 修复**: 正确提取 `cn/{slug}/{pkg}/download` 中文下载页
- **aiohttp HTML 检测**: 下载页 URL 自动降级 Playwright，不再保存 HTML 假 APK
- **Playwright 下载重写**: 三阶段策略 (监听下载事件 → 点击触发 → 页面直链提取) + `direct_url` 回传 aiohttp
- **前端三按钮**: 🌐详情页 (中文站跳转) + 🌐浏览器下载 (新标签页手动) + 📥点击下载 (系统自动)

### 🔧 Chromium 持久化
- **EXE 首次运行**: `launcher.py` 自动复制 Chromium 到 EXE 同目录，避免 Temp 被杀软拦截
- **查找优先级**: 环境变量 → 持久目录 → Temp 目录 → ms-playwright (dev)
- **EPIPE 消噪**: `browser_manager.py` 分步异常捕获 + 15s 超时 + 失败清理驱动进程

### 🧹 代码质量
- 前端 catch 块全部添加 `console.error` 日志
- `doFetchBatch` 补充 `resp.ok` 检查
- `download/manager.py` 线程竞态修复 (队列通信替代跨作用域共享变量)
- `config.py` 数值类型校验 + 负数拦截
